> Fa-fa-faaaa!

## Compatibility

- Linux
- Windows
- Mac

## Installing

Install globally:

    npm install -g mlg-horn

## Running

Run command:

    mlg-horn

Or import in NodeJS script:

    import mlgHorn from 'mlg-horn';

    mlgHorn();
